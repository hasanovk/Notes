Installation #: Written by KFH: karshi.hasanov@utoronto.ca #: Date: Oct. 10, 2010 #: Description: Ubuntu Apache Server #: Last Modified: March 19, 2018 => sudo apt-get install -y apache2 This will also create an Apache User and Group, both called "www-data" Installation Directory: /etc/apache2 "/etc/apache2/apache2.conf” # the main Apache2 configuration file "/etc/apache2/conf.d” # contains configuration files which apply globally to Apache2 "/etc/apache2/mods-available” # contains configuration files to both load modules and configure them "/etc/apache2/mods-enabled” # symlinks --> /etc/apache2/mods-available "/etc/apache2/sites-available” # this directory has configuration files for Apache2 Virtual Hosts. "/etc/apache2/sites-enabled” # symlinks --> "/etc/apache2/sites-available" Configuring your site => cd /etc/apache2/sites-available => sudo cp 000-default.conf osm.conf => sudo vim osm.conf <VirtualHost *:80> ServerAdmin karshi.hasanov@icloud.com ServerName osm.karshi.ca DocumentRoo
