Using the Ports Tree from the source code. # 2. Using the pkg tool which installs the binary versions. Method I: Installing the Applications uing the Ports # Require: The Ports Coleection. # The base system of FreeBSD includes the ports, so you can install # during the OS installation. # Optional (Recommended): Install one of the Ports Management Utility, # such as “potmaster”, “portupgrade” or lastest “synth”. # All these utilities are located under “/usr/ports/ports-mgmt” # I will be using the /usr/ports/ports-mgmt/synth. # Always make sure that you update the Ports tree before installing anything => sudo /usr/sbin/portsnap fetch update # Installing a software: As an example , we will install the lsof utility. => cd /usr/ports/sysutils/lsof => sudo make install clean
