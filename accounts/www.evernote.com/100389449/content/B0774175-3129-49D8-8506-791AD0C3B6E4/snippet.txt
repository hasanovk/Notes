Important Partitions #: Written by KFH: karshi.hasanov@utoronto.ca #: Date: March 24, 2018 #: Description: Sonim XP 770 Phone partitions. #: Last Modified: # There is an internal “sdcard” called “mmcblk0” # The following two commds show the same info about the mountes partitions: root@XP7700:/ # mount root@XP7700:/ # cat /proc/mounts # To see how many partitions the “mmcblk0” card has: root@XP7700:/ # cat /proc/partitions 179 0 15267840 mmcblk0 179 1 65536 mmcblk0p1 179 2 256 mmcblk0p2 179 3 256 mmcblk0p3 … 179 27 13238250 mmcblk0p27 179 32 4096 mmcblk0rpmb # The following command will show the filesystem by name : root@XP7700:/ # ls /dev/block/platform/msm_sdcc.1/by-name/ ( i.e. “boot”, “system”, “recovery”, “userdata” and “modem”) # How do we know which partition “boot” or “system” actually pointing to ? root@XP7700:/ # ls -l /dev/block/platform/msm_sdcc.1/by-name/boot boot -> /dev/block/mmcblk0p10 root@XP7700:/ # ls -l /dev/block/platform/msm_sdcc.1/by-name/system system -> /dev/blo
