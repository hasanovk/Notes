MySQL Data Types ------------------------------------------------ TINYINT from -128 —> 127 SMALLINT: -32,767 —> 32,768 MEDIUM INT: -8,388,608 —> 8,388,608 INT: (-2^31) -1 —> 2^31 BIGINT: (-2^63) -1 ——> 2^63 FLOAT: DOUBLE: CHAR: A character string with a fixed length VARCHAR: A character string with a length that is variable BLOD: Can contain 2^16 bytes of data ENUM: A character string that has a limited number of total values, which you must define. SET: A list of legal possible character strings. Unlike ENUM, it can contain multiple values in comparison to the one legal value with ENUM.
