NTP Server Setup Edit the file "/etc/ntp.conf" : server 0.ca.pool.ntp.org iburst server 1.ca.pool.ntp.org iburst server 2.ca.pool.ntp.org iburst # Create a drift file: => sudo touch /var/db/ntp.drift driftfile /var/db/ntp.drift # Create a log file: => sudo touch /var/log/ntp.log logfile /var/log/ntp.log Edit "/etc/rc.conf" and add the line: ntpd_enable="YES" If you did not setup your localtime during the installation, do so now: => sudo cp /usr/share/zoneinfo/America/Toronto /etc/localtime Update Time: => sudo ntpdate 0.ca.pool.ntp.org Start the deamon: => sudo /etc/rc.d/ntpd onestart # First time
