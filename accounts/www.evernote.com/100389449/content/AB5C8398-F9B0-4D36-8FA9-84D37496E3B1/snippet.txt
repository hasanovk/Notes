Hint: You can go to a specific line number while opening a file: vi +line_number filename.txt dw - delete a word ( from the current position to the end of the word) d$ - delete text from current position to the end of line D - delete from the current position to the end of the current line dG - delete from the current position to the end of the file d1G - delete from the current position to the beginig of the file <Shift> + d - delete characters from the current position to the end of a line u - Undo the Last command U - Undo the current line from All changed made on that line 0 - move the cursor to the beginning of the current line $ - move the cursor to the end of the current line CUT, COPY and PAST Note: You can use the DELETE options to CUT and then PAST with "pp" or "P" as shown below "yw" - copy a word "y$" - copy from the current position to the end of the line "y1G" - copy all lines from the current to the BEGINNING of the file "yG" - copy all lines from the current to the END
