# Written by KFH: karshi.hasanov@utoronto.ca # Date: August 2, 2017 # Last Modified: August 2, 2017 # Description: This HOWTO will describe how to setup a Git repository on FreeBSD 11.0 Step 1: Installing the Git # Always update the ports first => sudo /usr/sbin/portsnap fetch # Install the Git ( you can use the “pkg add “ command to install if you want) => cd /usr/ports/devel/git => sudo make install clean Step 2: Creating the dedicated user and group for Git repos => sudo pw groupadd -n git -g 6891 # Add the group name “git” with GUID = 6891 => sudo pw useradd -n git -u 6891 -g git -d /home/git -s /usr/local/libexec/git-core/git-shell -h - # Add the user name “git” with UID = 6891, HOME=/home/git, “s” allowed shell command. # Make sure that the directory exist and the owned by the “git user. => sudo mkdir /home/git => sudo chown -R git:git /home/git => sudo chmod 775 /home/git # We want to keep the repos in the /home/git/base folder: => sudo mkdir /home/git/base => sudo chmod 775 /ho
