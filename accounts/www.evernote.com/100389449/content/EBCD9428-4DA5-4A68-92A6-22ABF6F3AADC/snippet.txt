Installation for MAC OS X # Downloaded from: https://fastdl.mongodb.org/osx/mongodb-osx-ssl-x86_64-3.6.4.tgz # Website: https://www.mongodb.com Step 1: Download and extract the file "mongodb-osx-ssl-x86_64-3.6.4.tgz". Step 2: Create a folder for the MongoDB databases and log files: => mkdir /Volumes/Data/KFH/KFH_Data/Databases/MongoDB => mkdir /Volumes/Data/KFH/KFH_Data/Databases/MongoDB/db => mkdir /Volumes/Data/KFH/KFH_Data/Databases/MongoDB/log Step 3: Copy the files from the exatrcated folder to the new location: => cp -R -n /path/to/extracted/mongodb-osx-x86_64-3.6.4 \ /Volumes/Data/KFH/KFH_Data/Databases/MongoDB Step 4: Set the PATH for the MongoDB “bin” folder so the executable files are on our path. # In the “~/.bashrc” set PATH PATH=${PATH}:/Volumes/Data/KFH/KFH_Data/Databases/MongoDB/bin Step 5: Initializing the Server. Create a script file and put the folllowing lines: mongod \ --port 8102 \ --bind_ip_all \ --dbpath /Volumes/Data/KFH/KFH_Data/Databases/MongoDB/db \ --directo
