phpMyAdmin Installation on FreeBSD phpMyAdmin (MySQL or MariaDB ) => cd /usr/ports/databases/phpmyadmin/ => sudo make install clean This will install the phpMyAdmin into /usr/local/www/phpMyAdmin Edit config.inc.php to suit your needs. Edit the Apache main config file /usr/local/etc/apache24/httpd.conf and add the lines inside the <IfModule alias_module> directory: # phpMyAdmin Access Alias /phpmyadmin /usr/local/www/phpMyAdmin <Directory /usr/local/www/phpMyAdmin> Options None AllowOverride Limit Require local Require ip 192.168.100. allow from these subfolders to use the phpMyAdmin </Directory> Note: You might need: => sudo chmod 644 /usr/local/www/phpMyAdmin/config.inc.php
