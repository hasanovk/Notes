# Written by KFH: karshi.hasanov@utoronto.ca # Date: Feb 18, 2009 # Last Modified: August 6, 2017 # Description: Installing the Apache Web Server Apache 2.4 Server on FreeBSD Installation => cd /usr/ports/www/apache24 => sudo make install (leave the default options). If this is not your first installation, you might need to clean the old one: => sudo make clean && sudo make rmconfig Add the line apache24_enable='YES' in your /etc/rc.conf file Configuring the main httpd.conf File Open the /usr/local/etc/apache24/httpd.conf file and check the following settings: ServerAdmin karshi.hasanov@utoronto.ca DocumentRoot “ /opt/hosting/www.karshi.ca” # Default location of website <Directory "/opt/hosting/www.karshi.ca "> # These are defaults. Options Indexes FollowSymLinks AllowOverride None Require all granted </Directory> Initial Testing # Create a test file under the DocumentRoot folder # ( /opt/hosting/www.karshi.ca/index.html) # Restart the server: .=> sudo /usr/local/etc/rc.d/apache24 ones
