Creating a bootable ZFS Filesystem Boot FreeBSD install DVD or USB Memstick Choose Fixit option in sysinstall Create GPT Disks Fixit# gpart create -s gpt ad0 Fixit# gpart create -s gpt ad1 Fixit# gpart create -s gpt ad2 Create the boot, swap and zfs partitions Create 3 partitions on drives ad0, ad1 and ad2. The first partition contains the gptzfsboot loader which is able to recognize and load the loader from a ZFS partition. The second partition is a 4 GB swap partition. The third partition is the partition containing the zpool (60GB). Fixit# gpart add -b 34 -s 128k -t freebsd-boot ad0 Fixit# gpart add -s 4G -t freebsd-swap -l swap0 ad0 Fixit# gpart add -s 60G -t freebsd-zfs -l disk0 ad0 Fixit# gpart add -b 34 -s 128k -t freebsd-boot ad1 Fixit# gpart add -s 4G -t freebsd-swap -l swap1 ad1 Fixit# gpart add -s 60G -t freebsd-zfs -l disk1
