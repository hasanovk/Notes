jQuery -------------------------------------------------------------------------- <html> <head> <title> My test page </title> <!-- insert the script to reference jQuery here --> <script type="text/javascript" src="lib/jquery-2.1.4.js"> </script> </head> </html> --------------------------------------------------------------------------- Loading Web Pages Typically without jQuery, code that you want to execute when the page is loaded is written like this: function runOnLoad() { alert("My page is just loaded!"); } window.online = runOnLoad; The "unload" event only fires after all of the page content has downloaded, including images. In jQuery you can load the pages like this: Method 1: jQuery(document).ready(function() { alert("this works!); }); Method 2: (better): /* "$" is an alias to "jQuery" */ $(document).ready(function() { alert("this works too !); }); Method 3: (even better): /* "$" is an alias to "jQuery"+ "ready" */ $(function() { alert("this works as well !); }); jQuery Selector
