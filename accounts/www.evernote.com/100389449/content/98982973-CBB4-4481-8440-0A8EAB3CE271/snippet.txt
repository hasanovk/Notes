Add thefollowing line to /etc/fstab: linprocfs /compat/linux/proc linprocfs rw 0 0 => sudo mount /compat/linux/proc". Some programs need linsysfs mounted on /compat/linux/sys. Add the following line to /etc/fstab: linsysfs /compat/linux/sys linsysfs rw 0 0 => sudo mount /compat/linux/sys The Sublime 3 needs tmpfs mounted on /compat/linux/dev/shm. Add the following line to /etc/fstab: tmpfs /compat/linux/dev/shm tmpfs rw,mode=1777 0 0 => sudo mount /compat/linux/dev/shm ——————————————————————— # Edit “/etc/rc.conf” and make sure that the Linux support is enabled: linux_enable="YES" # Now you can reboot or manually the Linux modules “linux.ko” and “linux64.ko” # I believe the “linux64.ko” will be au
