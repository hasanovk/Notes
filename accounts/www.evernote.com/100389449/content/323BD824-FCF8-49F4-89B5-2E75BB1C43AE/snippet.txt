Package Management Install / Remove packages: => sudo apt-get install <pkg_name> ( you add "-y" after "apt-get" to say "Yes" to all) => sudo apt-get remove <pkg_name> Update/Upgrade => sudo apt-get update # To resynchronize the packages index files from their sources => sudo apt-get upgrade # To install newest versions of all packages currently installed Search Commands: => apt-cache search <pkg_name> even better => dpkg -l *<pkg_name>* ( "-l" is small "L") Show a package description: => apt-cache show <pkg_name> => dpkg --print-avail <pkg_name> Show what files listed in the package: => dpkg -L <pkg_name> Remove unused packages => sudo apt-get auto remove -y
