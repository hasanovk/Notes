Git Repository on FreeBSD #: Written by KFH: karshi.hasanov@utoronto.ca #: Date: May 25, 2018 #: Description: Setting up the Git Server on FreeBSD 11.1 #: Last Modified: -- # Update the packages => sudo portsnap fetch update # Install dependencies: => sudo pkg install -y xmlcatmgr => sudo pkg install -y libatomic_ops => sudo pkg install -y perl5 # Install the Git => cd /usr/ports/devel/git => sudo make install clean # Create a Git user: => sudo pw groupadd -n git -g 9418 => sudo pw useradd -n git -u 9418 -g git -c git -d /git -s /usr/local/libexec/git-core/git-shell -h - # Create the Git directory: => sudo mkdir /git => sudo chown git:git /git/ => sudo chmod 755 /git => sudo mkdir /git/base/ => sudo chown git:git /git/base/ => sudo chmod 775 /git/base/ # Create the SSH Key # You can dump your public key to the “authorized_keys” to auto login. => sudo mkdir /git/.ssh/ => sudo chmod 700 /git/.ssh/ => sudo touch /git/.ssh/authorized_keys => sudo chmod 600 /git/.ssh/authorized_keys => sudo
