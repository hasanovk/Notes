Install the x11vnc Server => cd /usr/ports/net/x11vnc => sudo make install clean # Create a password which will be used to login => x11vnc -storepassword # The above command will create the password and stores in # the $USER/.vnc/password file # To start the server: => x11vnc -rfbauthb /home/khasanov/.vnc/passwd -forever -display :0 # Note: There is the space between “display” and “:0” (‘0’ means first display) Using the VNC Server from the client side # I am using the Mac OS X on iMac Go -> Connect To Server.. Then you need to enter the password you’ve created for the x11nvc.
