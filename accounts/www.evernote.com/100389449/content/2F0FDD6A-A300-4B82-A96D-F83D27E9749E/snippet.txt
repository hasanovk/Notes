How to setup the Static IP Edit "/etc/rc.conf" ifconfig_xn0="inet 10.10.10.150 netmask 255.255.255.0" defaultrouter="10.10.10.10" Edit "/etc/resolv.conf" nameserver 192.168.21 nameserver 10.10.10.120 How to "Start","Stop" or "Restart" interfaces: => sudo /etc/rc.d/netif start | stop | restart To see the list of "Up/Down" interfaces: => ifconfig -u | -d To update the "Routing" table: => /etc/rc.d/routing restart Note: Over SSH you must use "&&" to avoid disconnection: => sudo /etc/rc.d/netif restart && sudo
